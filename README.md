# NC VIDEO CONVERTER
- Aplicativo convertidor de video desarrollado con Web Components, Node, Rust, WebAssembly y la libreria FFmpeg 

<br><hr>

## TECNOLOGIAS
* Frontend: Web Components (LIT)
* Backend: NODE, RUST
* WebAssembly
* Libreria FFmpeg
